# Diamond Nova

야구 데이터 분석 및 시각화 프로젝트입니다. MLB 데이터를 활용하여 타격 및 투구 통계를 분석하고, 웹 대시보드를 통해 직관적으로 확인할 수 있습니다.

## 프로젝트 구조

- `batting-최종.csv`: 타격 데이터 파일
- `pitching-dashbord.csv`: 투구 대시보드 데이터 파일
- `pitching.csv`: 투구 데이터 파일
- `frontend/`: Next.js 기반 프론트엔드 애플리케이션

## 기술 스택

- **프론트엔드**: Next.js, TypeScript, Tailwind CSS
- **데이터**: CSV 파일 기반 분석
- **배포**: GitHub Pages 또는 Vercel 추천

## 설치 및 실행

### 프론트엔드 실행

1. `frontend/` 폴더로 이동:
   ```bash
   cd frontend
   ```

2. 의존성 설치:
   ```bash
   npm install
   ```

3. 개발 서버 실행:
   ```bash
   npm run dev
   ```

4. 브라우저에서 `http://localhost:3000` 접속

### 데이터 분석

CSV 파일들은 Python이나 R을 사용하여 추가 분석 가능합니다. 예시:
- Pandas로 데이터 로드 및 분석
- Matplotlib/Seaborn으로 시각화

## 사용법
